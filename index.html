Wizard Penguin Histogram

I used excel to make this histogram for Wizard Penguin. First, I entered the data into Excel columns. I then divided the values by their max values so that a percentage could be calculated.

Finally, I averaged the percentages in each column, converted the decimals to percentages, and used those values in my histogram.

Note: I did not find the average of the final grade because there was only one data value used.
